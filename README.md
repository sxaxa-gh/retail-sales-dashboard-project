# Retail Sales Dashboard Project

## Project Overview
This project analysed retail sales data to identify regional sales performance, product profitability trends, monthly sales patterns, and top-performing products. The aim of the project was to demonstrate how data analytics and visualisation techniques can support business decision-making within a retail environment.

## Dataset
The project used the publicly available Superstore Sales dataset sourced from Kaggle. The dataset contains transactional retail sales information including product categories, sales values, profits, customer regions, and order dates.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualisation Charts
- GitHub

## Data Engineering
The dataset was prepared and organised within Excel before analysis was performed. Data preprocessing included structuring data into tables, aggregating sales and profit values, and organising information into categories suitable for visual analysis.

## Data Analytics
The analysis focused on identifying:
- Sales performance across regions
- Profit contribution by product category
- Monthly sales trends over time
- Top-performing products by sales value

The project used exploratory data analysis techniques to identify trends, patterns, and business insights from the retail dataset.

## Visualisations

### Sales by Region
A horizontal bar chart was used to compare sales performance across geographical regions. The analysis showed that the West region generated the highest sales performance.

### Profit by Product Category
A pie chart was used to visualise the proportion of profit generated by different product categories. Technology products generated the highest overall profit contribution.

### Monthly Sales Trend
A line chart was developed to analyse changes in sales performance over time. The visualisation highlighted fluctuations in monthly sales activity and seasonal demand patterns.

### Top Products by Sales
A column chart was created to compare the highest-performing products by sales revenue. Canon Copier products generated the highest sales value within the dataset.

## Business Impact
This project demonstrates how retail organisations can use data analytics and visualisation techniques to improve operational decision-making, monitor sales performance, identify profitable product areas, and support strategic planning.

## Ethical Considerations
The project used a publicly available dataset and therefore did not contain sensitive personal information. However, ethical considerations relating to data privacy, responsible reporting, and accurate interpretation of results remain important within retail analytics projects.
